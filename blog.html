Variable

Global Scope: A variable (var, let, const) declare outsite of this{} bracket it's global scope. In js code anywhere can access global scope declared variable.

Function Scope: If a variable (var, let, const) declare in the function than it can't be access outsite of function.

Block Scope: If a variable declare in this {} bracket than it can't be access outsite of this {} bracket. let and const are block scope variable.

Null and Undefined

Undefined: We declare a variable anywhere in JS without assign any value then we get output as a undefine

Null: We declare a variable and want to show empty data then we can assign Null as a value.

What do you mean by REST API?

REST API: REST API can be a standard used as an architectural means of building a network-based software system.It's not tools, protocall and library. It's web services communication between computer and web system.
